## Decision Authority Matrix (RACI)

| Decision Type | Sponsor | Product Owner | Tech Lead | Dev Team |
|---------------|---------|---------------|-----------|----------|
| Scope changes | A | R | C | I |
| Technical approach | I | C | R | C |
| Timeline changes | A | R | C | I |
| Resource allocation | A | R | C | I |
| Feature prioritization | I | R | C | I |
| Go/No-go decisions | A | R | C | I |

**Legend**: R = Responsible, A = Accountable, C = Consulted, I = Informed
