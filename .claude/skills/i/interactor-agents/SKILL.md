# Interactor AI Agents Skill

**Documentation:** `docs/i/interactor-docs/integration-guide/04-ai-agents.md`

## When to Use

- Building conversational AI chat interfaces
- Creating tool-calling assistants that invoke custom functions
- Connecting AI to databases for natural language queries
- Building customer support bots with domain knowledge
- Creating internal AI assistants for operations

## Prerequisites

- Interactor authentication configured (see `interactor-auth` skill)
- Webhook endpoint for tool callbacks (optional, for custom tools)
- Database with network access from Interactor (optional, for data sources)

## Instructions

Read the full documentation at the source path above for:
- Assistant configuration and management
- Room (chat session) and message handling
- Custom tool definitions and webhooks
- Data source connections (databases, APIs)
